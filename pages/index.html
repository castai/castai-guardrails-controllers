CAST AI Guardrails Controllers

Helm chart repository for castai-guardrails-controllers.

Usage

helm repo add castai-guardrails https://castai.github.io/castai-guardrails-controllers
helm repo update

Available charts

helm install castai-pdb-controller castai-guardrails/castai-pdb-controller
helm install castai-tsc-controller castai-guardrails/castai-tsc-controller
helm install castai-jvm-probe-controller castai-guardrails/castai-jvm-probe-controller

Browse versions

helm search repo castai-guardrails --versions
